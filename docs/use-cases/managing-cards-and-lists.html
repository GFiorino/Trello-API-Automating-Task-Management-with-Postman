Trello API: Managing Cards and Lists

This guide explains how to manage cards and lists using the Trello API.

Step 1: Retrieve the Card ID

To move or update a specific card, retrieve its ID using a GET request:

GET https://api.trello.com/1/boards/{boardID}/cards

Step 2: Move a Card to Another List

Move a card by making a PUT request:

PUT https://api.trello.com/1/cards/{cardID}?idList={newListID}

Step 3: Add a Label to a Card

Retrieve available labels and assign one using the following request:

POST https://api.trello.com/1/cards/{cardID}/idLabels

Step 4: Archive a List

Archive a list by setting the closed parameter to true:

PUT https://api.trello.com/1/lists/{listID}?closed=true
