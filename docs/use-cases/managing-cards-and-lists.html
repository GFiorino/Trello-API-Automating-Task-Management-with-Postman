Trello API: Managing Cards and Lists

This section demonstrates how to manage cards and lists using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

Step 1: Retrieve the Card ID

To move or update a specific card, you first need to retrieve its ID. In Postman, make a GET request to the following endpoint using your API Key and Token: 

GET https://api.trello.com/1/boards/{boardID}/cards?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

Step 2: Move a Card to Another List

Once you have the Card ID, move the card to a new list by making a PUT request: 

PUT https://api.trello.com/1/cards/{cardID}
Content-Type: application/json

{
  "idList": "{newListID}"
}

Step 3: Add a Label to a Card

Retrieve available labels and add a label to your card: 

POST https://api.trello.com/1/cards/{cardID}/idLabels
Content-Type: application/json

{
  "value": "{labelID}"
}

Step 4: Archive a List

Archive a list by setting the closed parameter to true: 

PUT https://api.trello.com/1/lists/{listID}
Content-Type: application/json

{
  "closed": true
}
