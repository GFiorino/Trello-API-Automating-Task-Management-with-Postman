Trello API (v1): Managing Cards and Lists

This guide explains how to manage cards and lists using the Trello API. Follow the steps below to retrieve card IDs, move cards between lists, add labels to cards, and archive lists.

Step 1: Retrieve the Card ID

To perform operations on a specific card, first retrieve its unique ID. Make a GET request: 

GET https://api.trello.com/1/boards/{boardID}/cards

The response will contain details of all cards on the board, including their unique IDs.

Step 2: Move a Card to Another List

To move a card, update its idList by making a PUT request: 

PUT https://api.trello.com/1/cards/{cardID}?idList={newListID}

Replace {cardID} with the actual card ID and {newListID} with the target list ID.

Step 3: Add a Label to a Card

To assign a label to a card, first retrieve the available labels, then use a POST request to add a label: 

POST https://api.trello.com/1/cards/{cardID}/idLabels

Ensure you replace {cardID} with the actual card ID.

Step 4: Archive a List

To archive a list, update the list's closed parameter by making a PUT request: 

PUT https://api.trello.com/1/lists/{listID}?closed=true

Replace {listID} with the actual ID of the list you wish to archive.
