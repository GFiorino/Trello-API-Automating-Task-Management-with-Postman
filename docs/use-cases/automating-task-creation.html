Trello API: Automating Task Creation

This guide explains how to automate task creation using the Trello API.

Step 1: Obtain Your API Key

Retrieve your API Key from the Trello API settings page. Ensure you add "https://www.postman.com" to the Allowed Origins field.

Step 2: Generate a Token

Click "Generate Token" on the API Key page and copy it for later use.

Step 3: Retrieve the List ID

Use the following GET request to retrieve the List ID where tasks will be created:

GET https://api.trello.com/1/boards/{boardID}/lists?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

Step 4: Create a Task Using a POST Request

Use the following request to create a task:

POST https://api.trello.com/1/cards
Content-Type: application/json

{
  "name": "Automated Task Example",
  "desc": "Task created via API automation.",
  "idList": "{LIST_ID}",
  "pos": "top",
  "due": "2024-12-10T09:00:00.000Z",
  "key": "YOUR_API_KEY",
  "token": "YOUR_ACCESS_TOKEN"
}

Expected Outcome

A new task titled "Automated Task Example" will appear in the specified Trello list.
