Trello API (v1): Automating Task Creation

This guide provides a step-by-step approach to automate task creation using the Trello API. Follow the instructions and refer to the screenshots to successfully automate task creation in your Trello boards.

Step 1: Retrieve API Key

Navigate to the Trello API Key Page and copy your API key. Make sure to add "https://www.postman.com" to the Allowed Origins field.

Step 2: Generate API Token

On the API Key page, click "Generate Token" to obtain your API token. This token will allow your application to authenticate and interact with the Trello API.

Step 3: Locate List ID

Identify the target list where tasks will be created. You can find the List ID in the board URL or by making an API call.

Step 4: Create a Task via API

Send a POST request to add a new task to your chosen list. Use the following JSON structure, replacing {LIST_ID} with your actual List ID. 

{
  "name": "Automated Task",
  "desc": "Task created via API automation",
  "idList": "{LIST_ID}",
  "pos": "top",
  "due": "2024-12-10T09:00:00.000Z"
}

Expected Output

After the request is successfully executed, the new task should appear in the specified Trello list.
