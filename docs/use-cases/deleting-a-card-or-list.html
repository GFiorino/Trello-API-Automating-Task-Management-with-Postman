Trello API: Deleting a Card

This section demonstrates how to delete a card from your Trello board using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

To delete a card, you first need to retrieve its ID. Make a GET request to the following endpoint: 

https://api.trello.com/1/cards/{cardID}

The response will include the card's details, including its unique Card ID.

Once you have the Card ID, make a DELETE request to the following endpoint:

Replace {cardID} with the ID of the card you want to delete.

  1. Set the HTTP method to DELETE.
  2. Enter the endpoint, replacing {cardID} with the card's unique ID.
  3. Click Send to execute the request.
