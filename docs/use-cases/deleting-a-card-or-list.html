Trello API: Deleting a Card

This guide explains how to delete a card from your Trello board using the Trello API.

Step 1: Retrieve the Card ID

To delete a card, you first need to retrieve its ID. Use the following GET request:

GET https://api.trello.com/1/cards/{cardID}?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

Step 2: Delete the Card

Once you have the Card ID, use the following DELETE request:

DELETE https://api.trello.com/1/cards/{cardID}?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

Replace {cardID} with the ID of the card you want to delete.

  1. Set the HTTP method to DELETE.
  2. Enter the endpoint, replacing {cardID} with the correct ID.
  3. Click Send to execute the request.

Expected Outcome

The specified card will be successfully deleted from your Trello board. A 200 OK response confirms the deletion.
