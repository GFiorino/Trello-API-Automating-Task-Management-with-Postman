Trello API (v1): Deleting a Card

This guide explains how to delete a card from your Trello board using the API. Follow the steps below to retrieve the card ID and then delete the card.

Step 1: Retrieve the Card ID

Make a GET request to obtain the card ID: 

GET https://api.trello.com/1/cards/{cardID}

The response will include the card's unique ID.

Step 2: Delete the Card

Use the retrieved card ID in a DELETE request to remove the card: 

DELETE https://api.trello.com/1/cards/{cardID}

Replace {cardID} with the actual card ID.

Expected Output

Once the request is successfully processed, the card will be removed from Trello and you will receive a 200 OK response.
