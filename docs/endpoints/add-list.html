Trello API: Add List

This endpoint allows you to create a new list on an existing Trello board.

Request Details

  • Method: POST
  • Endpoint URL: https://api.trello.com/1/lists
  • Authentication: Requires API key and token as query parameters.

Required Parameters

Parameter Type Required Description
name String ✔ The name of the new list.
idBoard String ✔ The ID of the board where the list will be added.

Example Request

POST https://api.trello.com/1/lists
Content-Type: application/json

{
  "name": "New List Example",
  "idBoard": "12345abcdef",
  "pos": "top",
  "key": "your_api_key",
  "token": "your_access_token"
}

Common Errors

Error Possible Cause Solution
401 Unauthorized Invalid API key or token. Verify credentials on Trello API Key Page.
400 Bad Request Missing required parameters. Ensure idBoard and name are provided.
404 Not Found Invalid or non-existing board ID. Double-check the idBoard value.
