Trello API: Add Card

This endpoint allows you to add a new card to a specified list within a Trello board.

Request Details


POST https://api.trello.com/1/cards
Content-Type: application/json

{
  "name": "New Task",
  "desc": "This is a sample Trello card.",
  "idList": "LIST_ID",
  "pos": "top",
  "due": "2024-12-31T23:59:59.000Z"
}
