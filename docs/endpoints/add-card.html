Trello API (v1): Add Card

Use this endpoint to create a new card on a specified list within a Trello board. Ensure that you have already authenticated using your API key and token.

üìå Request Details

The following is the structure of the POST request for adding a new card. Replace LIST_ID with the ID of your target list.

cURL
Node.js
curl --request POST \
  --url "https://api.trello.com/1/cards?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN" \
  --header "Content-Type: application/json" \
  --data '{
    "name": "New Task",
    "desc": "This is a sample Trello card.",
    "idList": "LIST_ID",
    "pos": "top",
    "due": "2024-12-31T23:59:59.000Z"
}'
        
// Using fetch (Node 18+ or with node-fetch)
fetch("https://api.trello.com/1/cards?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    name: "New Task",
    desc: "This is a sample Trello card.",
    idList: "LIST_ID",
    pos: "top",
    due: "2024-12-31T23:59:59.000Z"
  })
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));
