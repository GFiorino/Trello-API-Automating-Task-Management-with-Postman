Trello API: Create Board

The Create Board endpoint allows you to create a new board in Trello. You need to provide a name for the board and, optionally, other parameters like default lists and organization ID.

üîó Request Details

  • Method: POST
  • Endpoint URL: https://api.trello.com/1/boards
  • Authentication: Requires key and token as query parameters.

Parameter Type Description
name String The name of the new board.
