Trello API: Create Board

The Create Board endpoint allows you to create a new board in Trello.

Request Details

  • Method: POST
  • Endpoint URL: https://api.trello.com/1/boards
  • Authentication: Requires API key and token as query parameters.

Required Parameters

Parameter Type Required Description
name String ✔ The name of the new board.

Example Request

POST https://api.trello.com/1/boards
Content-Type: application/json

{
  "name": "New Board Example",
  "defaultLists": true,
  "idOrganization": "60d21b4967d0d810b8f8b456",
  "key": "YOUR_API_KEY",
  "token": "YOUR_ACCESS_TOKEN"
}

Common Errors

Error Possible Cause Solution
401 Unauthorized Invalid API key or token. Verify credentials on Trello API Key Page.
400 Bad Request Missing required parameters. Ensure name is provided.
404 Not Found Invalid idOrganization value. Double-check the idOrganization.
