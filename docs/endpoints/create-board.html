Trello API (v1): Create Board

The Create Board endpoint allows you to create a new board in Trello. Ensure you have authenticated using your API key and token before using this endpoint.

Request Details

  • Method: POST
  • Endpoint URL: https://api.trello.com/1/boards
  • Authentication: Requires API key and token as query parameters.

Parameters

Parameter Type Required Description
name String ✔ The name of the new board.
defaultLists Boolean ✖ If set to true, default lists (To Do, Doing, Done) are created.
idOrganization String ✖ The ID of the organization under which to create the board.

Example Request

The following is an example POST request to create a new board. Replace YOUR_API_KEY, YOUR_ACCESS_TOKEN, and other placeholders with your actual values. 


POST https://api.trello.com/1/boards
Content-Type: application/json

{
  "name": "New Board Example",
  "defaultLists": true,
  "idOrganization": "60d21b4967d0d810b8f8b456",
  "key": "YOUR_API_KEY",
  "token": "YOUR_ACCESS_TOKEN"
}

Common Errors

Error Possible Cause Solution
401 Unauthorized Invalid API key or token. Verify your credentials on the Trello API Key Page.
400 Bad Request Missing required parameters. Ensure the name parameter is provided.
404 Not Found Invalid idOrganization value. Double-check the provided organization ID.
