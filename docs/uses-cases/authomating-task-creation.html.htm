Trello API: Automating Task Creation

This guide explains how to automate task creation using the Trello API, with a structured step-by-step approach.

Step 1: Retrieve API Key

Go to Trello API Key Page and copy your API key. Add "https://www.postman.com" to the Allowed Origins field.

Step 2: Generate API Token

Click "Generate Token" on the API Key page to obtain your authentication token.

Step 3: Locate List ID

Find the List ID where tasks will be created using the board URL or API call.

Step 4: Create a Task via API

Send a POST request to add a task using the following JSON body:

{
  "name": "Automated Task",
  "desc": "Task created via API automation",
  "idList": "{LIST_ID}",
  "pos": "top",
  "due": "2024-12-10T09:00:00.000Z"
}

Expected Output

The task should now be visible in the specified Trello list.
