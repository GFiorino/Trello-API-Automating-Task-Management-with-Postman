Trello API: Deleting a Card

This guide explains how to delete a card from your Trello board using the API.

Step 1: Retrieve the Card ID

Make a GET request to obtain the card ID:

GET https://api.trello.com/1/cards/{cardID}

The response will include the card's unique ID.

Step 2: Delete the Card

Use the card ID in a DELETE request:

DELETE https://api.trello.com/1/cards/{cardID}

Replace {cardID} with the actual card ID.

Expected Output

The card is removed from Trello, and you receive a 200 OK response.
