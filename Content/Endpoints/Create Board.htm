Create Board

The Create Board endpoint allows you to create a new board in Trello. You need to provide a name for the board and, optionally, other parameters like default lists and organization ID.

Request Details

  • Method: POST
  • Endpoint URL: https://api.trello.com/1/boards
  • Authentication: Requires key and token as query parameters.

Required Parameters

Parameter Type Description
name String The name of the new board.

Optional Parameters

Parameter Type Description
defaultLists Boolean Whether to include default lists (To Do, Doing, Done).
idOrganization String The ID of the organization to which the board will belong.

Example Request

POST https://api.trello.com/1/boards
Content-Type: application/json

{
  "name": "New Board Example",
  "defaultLists": true,
  "idOrganization": "60d21b4967d0d810b8f8b456",
  "key": "YOUR_API_KEY",
  "token": "YOUR_ACCESS_TOKEN"
}

Example Response

{
  "id": "60d21b4967d0d810b8f8b456",
  "name": "New Board Example",
  "url": "https://trello.com/b/12345abcd/NewBoardExample",
  "prefs": {
    "permissionLevel": "private",
    "voting": "disabled",
    "comments": "members",
    "invitations": "members",
    "selfJoin": true,
    "cardCovers": true,
    "background": "blue",
    "calendarFeedEnabled": false
  }
}

Common Errors

Error Possible Cause Solution
401 Unauthorized Invalid API key or token. Verify your API key and token are correct.
400 Bad Request Missing required parameters (e.g., name). Ensure all required parameters are provided.
404 Not Found Invalid idOrganization value. Verify the idOrganization is valid.

Retrieving the idOrganization

To retrieve the idOrganization, use the https://api.trello.com/1/members/me/organizations endpoint with your API key and token. This will return all organizations associated with your account.

  1. Log in to Trello and ensure you are a member of an organization.
  2. Use the endpoint above to retrieve the ID of the desired organization.
