Automating Task Creation

This section demonstrates how to automate task creation using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

Step 1: Obtain Your API Key

Retrieve your API Key from Trello's API settings page.

Figure 1: Retrieving the API Key.

Tip: Add https://www.postman.com to the Allowed Origins field in your API settings to ensure Postman requests are accepted.

Step 2: Generate a Token

Click Generate Token on the API page and copy it for use in Postman. This token allows you to authenticate your requests securely.

Figure 2: Generating the API Token.

Step 3: Using Postman for API Requests

Open Postman and configure the request:

  1. Set the HTTP method to POST.
  2. Enter the API endpoint https://api.trello.com/1/cards.
  3. Add query parameters key and token.
  4. Use a JSON body to define the card's properties (e.g., name, desc, idList).
  5. Click Send to create the card.

Figure 3: Configuring the POST request in Postman.
