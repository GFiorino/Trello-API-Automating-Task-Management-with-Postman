Automating Task Creation

This section demonstrates how to automate task creation using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

Step 1: Obtain Your API Key

Navigate to your Trello account and retrieve your API Key from the API settings page. Ensure you add "https://www.postman.com" to the Allowed Origins field.

Figure 1: Retrieving the API Key from Trello's API settings page.

Step 2: Generate a Token

After obtaining your API Key, click the "Generate Token" link. Copy the generated token for use in Postman.

Figure 2: Generating and copying the Token for use in API requests.

Step 3: Retrieve the Short ID from the Trello Board

Navigate to your Trello board. Locate the short ID in the board's URL to reference the specific board or list.

Figure 3: Locating the Short ID in the Trello board URL.

Step 4: Use Postman to Retrieve the Full List ID

In Postman, make a GET request to the following endpoint with your API Key and Token: https://api.trello.com/1/boards/{shortID}/lists.

Figure 4: Using Postman to retrieve the List ID from Trello's API.

Step 5: Create a New Task Using POST

Using the retrieved List ID, create a new task by making a POST request in Postman. Follow these steps:

  1. Set the HTTP method to POST.
  2. Enter the following endpoint: https://api.trello.com/1/cards?key={APIKey}&token={Token}.
  3. Go to the Body tab in Postman, select raw, and set the format to JSON.
  4. Enter the following JSON object in the Body section:
{
  "name": "Automated Task Example",
  "desc": "This task was created as part of the API automation demonstration.",
  "idList": "{ListID}",
  "pos": "top",
  "due": "2024-12-10T09:00:00.000Z"
}

Click the Send button to execute the request.

Figure 5: Configuring the POST request in Postman to create a new task.

Once the request is successfully sent, a new task will be created in Trello under the specified list.

Step 6: Verify the Task in Trello

Check your Trello board to ensure the task has been created successfully. The task should appear in the specified list with the provided details.

Figure 6: Verifying the newly created task on the Trello board.

Expected Outcome

A new task titled "Automated Task Example" will appear in the specified list on your Trello board, complete with a description, due date, and position settings.
