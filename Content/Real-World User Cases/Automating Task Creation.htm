Trello API: Automating Task Creation

This section demonstrates how to automate task creation using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

Step 1: Obtain Your API Key

Navigate to your Trello account and retrieve your API Key from the API settings page. Ensure you add "https://www.postman.com" to the Allowed Origins field.

Figure 1: Retrieving the API Key from Trello's API settings page.

Note: Only share the token with trusted applications. Do not expose it publicly for security reasons.

Step 2: Generate a Token

Generate a token by clicking the "Generate Token" link on the API Key page and copy it for later use.

Figure 2: Generating a token for API authentication.

Step 3: Retrieve the Short ID

Find the Short ID in the Trello board URL for referencing specific lists and tasks.

Figure 3: Retrieving the Short ID from the Trello board URL.

Step 4: Create a Task Using POST Request

Use your API Key, Token, and List ID to create a task with the following JSON body:

{
  "name": "Automated Task Example",
  "desc": "Task created as part of the API automation.",
  "idList": "{ListID}",
  "pos": "top",
  "due": "2024-12-10T09:00:00.000Z"
}

Figure 4: Creating a task using a POST request in Postman.

Expected Outcome

A new task titled "Automated Task Example" will appear in the specified list on your Trello board.

Figure 5: Verifying the created task in Trello.
