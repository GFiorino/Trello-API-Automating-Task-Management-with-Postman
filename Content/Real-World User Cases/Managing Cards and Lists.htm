Managing Cards and Lists

This section demonstrates how to manage cards and lists using the Trello API. Follow the steps below to understand the process, illustrated with screenshots.

Step 1: Retrieve the Card ID

To move or update a specific card, you first need to retrieve its ID. In Postman, make a GET request to the following endpoint using your API Key and Token:

https://api.trello.com/1/boards/{boardID}/cards

The response will include the card's details, including the ID.

Figure 1: Retrieving the card ID using a GET request in Postman.

Step 2: Move a Card to Another List

Once you have the Card ID, you can move the card to a different list by making a PUT request. Use the following endpoint and include the new List ID as a parameter: 

https://api.trello.com/1/cards/{cardID}?idList={newListID}
  1. Set the HTTP method to PUT.
  2. Enter the endpoint, replacing {cardID} and {newListID} with the respective IDs.
  3. Click Send to execute the request.

Step 3: Add a Label to a Card

To organize your cards, you can add labels. First, retrieve the available labels for your board by making a GET request to: 

https://api.trello.com/1/boards/{boardID}/labels

Then, add a label to a card by making a POST request to: 

https://api.trello.com/1/cards/{cardID}/idLabels

Include the Label ID in the request body.

  1. Retrieve the Label ID using the GET endpoint above.
  2. Set the HTTP method to POST.
  3. Include the Label ID in the body of your request.
  4. Click Send to apply the label.

Step 4: Archive a List

To clean up your board, you may want to archive an entire list. Use the following endpoint and set the closed parameter to true: 

https://api.trello.com/1/lists/{listID}
  1. Set the HTTP method to PUT.
  2. Enter the endpoint, replacing {listID} with the ID of the list you want to archive.
  3. Click Send to archive the list.

Expected Outcome

By completing the steps above, you will have successfully managed cards and lists on your Trello board. This includes moving cards between lists, adding labels, and archiving lists for better organization.
