Setting Up API Authentication

To access the Trello API, you must authenticate your requests. Trello uses an API key and a token-based system to ensure secure communication between your application and its servers.

Steps to Authenticate

  1. Generate an API Key:

    • Log in to your Trello account.
    • Visit the Trello API Key page at https://trello.com/app-key.
    • Copy the API Key provided.

  2. Generate an Access Token:

    • Scroll down on the API Key page and click the link to generate a token.
    • Grant the required permissions when prompted.
    • Copy the access token for use in your API requests.

  3. Include the Key and Token in Requests:

    • Add your API key and token as query parameters in your request URLs.
    https://api.trello.com/1/resource?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

Example Request

GET https://api.trello.com/1/members/me?key=YOUR_API_KEY&token=YOUR_ACCESS_TOKEN

This example retrieves information about the authenticated user.

Common Errors

Error Possible Cause Solution
401 Unauthorized Invalid API key or token. Verify that your API key and token are correct.
400 Bad Request Malformed or missing query parameters. Ensure the key and token are included in the request URL.
